# ToDoList_AI

โปรแกรม CLI แบบง่ายสำหรับจัดการ To-Do list (ภาษาไทย)

**คำอธิบายโปรเจค**
- โปรแกรมเป็นหน้าตา CLI ที่ให้ผู้ใช้เพิ่ม ดู แก้ไข และลบงาน
- ข้อมูลงานเก็บอยู่ในไฟล์ `tasks.json` (JSON) เพื่อให้ข้อมูลคงอยู่ระหว่างการใช้งาน

**ข้อกำหนด (Prerequisites)**
- Python 3.7 ขึ้นไป

**ติดตั้ง / เตรียมตัว**
1. เปิด PowerShell หรือ terminal ของคุณ
2. ไปยังโฟลเดอร์โปรเจค:

```powershell
cd ...\Ai_code\ToDoList_AI
```

(ถ้าคุณได้ clone หรือดาวน์โหลดโปรเจคไว้ที่อื่น ให้ปรับพาธให้ตรง)

**รันโปรแกรม**

```powershell
python ...\Ai_code\ToDoList_AI\main.py
```

เมื่อโปรแกรมเริ่มจะโหลด `tasks.json` ถ้ามี และเมื่อออกจากโปรแกรม (เลือกเมนู 5) โปรแกรมจะบันทึกข้อมูลลง `tasks.json` ให้โดยอัตโนมัติ

**เมนูหลัก (ตัวอย่างการใช้งาน)**
- 1. เพิ่มงานใหม่
  - ใส่ `ชื่อเรื่อง`, `รายละเอียด`, `วันครบกำหนด` (รูปแบบตัวอย่าง: `YYYY-MM-DD`)
- 2. ดูงานทั้งหมด
  - แสดงรายการที่มีพร้อมหมายเลข (index), ชื่อ, วันครบกำหนด, และสถานะ
- 3. แก้ไขงาน
  - เลือกหมายเลขงาน (index) ที่จะแก้ จากนั้นสามารถแก้ `title`, `description` และ `completed` (y/n)
- 4. ลบงาน
  - เลือกหมายเลขงาน (index) แล้วยืนยันการลบด้วย `y` หรือ `n`
- 5. ออกจากโปรแกรม
  - โปรแกรมจะบันทึกข้อมูลก่อนออก

**ตัวอย่างเซสชันสั้น ๆ**
1) รันโปรแกรม
2) เลือก `1` -> เพิ่มงานใหม่ (ใส่ชื่อและรายละเอียด)
3) เลือก `2` -> ตรวจสอบรายการงาน
4) เลือก `3` -> แก้ไขงานหมายเลข `1` (แก้ชื่อหรือเปลี่ยนสถานะเป็นเสร็จแล้ว)
5) เลือก `4` -> ลบงานหมายเลข `1` (ตอบ `y` เพื่อยืนยัน)
6) เลือก `5` -> ออกจากโปรแกรมและบันทึก

**ไฟล์ที่สำคัญ**
- `main.py` : โค้ดหลักของโปรแกรม
- `tasks.json` : ไฟล์ข้อมูลเก็บรายการงาน (สร้างโดยอัตโนมัติเมื่อบันทึกครั้งแรก)

**ข้อเสนอแนะเพิ่มเติม (ถ้าต้องการ)**
- เพิ่มการค้นหา / กรองงานตามวันครบกำหนด
- เพิ่มการจัดเก็บเป็นฐานข้อมูลขนาดเล็ก (SQLite) ถ้าต้องการความทนทานสูงขึ้น

ถ้าต้องการ ผมสามารถช่วยเพิ่มตัวอย่างการใช้งานจริง (screenshot หรือตัวอย่าง input/output) หรือช่วย commit และ push ให้ได้ครับ
